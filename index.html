question 1

input
第一行一个整数 T，表示有 T 组数据。
每组数据第一行一个正整数 N（N<=50000）,表示敌人有N个工兵营地，接下来有N个正整数,第i个正整数ai代表第i个工兵营地里开始时有 ai 个人（1<=ai<=50）。
接下来每行有一条命令，命令有4种形式：
(1) Add i j,i和j为正整数,表示第i个营地增加j个人（j不超过30）
(2) Sub i j,i 和 j 为正整数,表示第 i 个营地减少 j 个人（j 不超过30）;
(3) Query i j,i 和 j 为正整数,i<=j，表示询问第 i 到第 j 个营地的总人数;
(4) End 表示结束，这条命令在每组数据最后出现;
output
对第 i 组数据,首先输出“Case i:”和回车,
对于每个Query询问，输出一个整数并回车,表示询问的段中的总人数,这个数保持在int以内

浅谈·梯度下降

发表于 2023-11-06 更新于 2024-01-20 分类于 机器学习

为什么需要梯度下降？

梯度下降是一种基于迭代的优化算法，用于最小化函数的值。它是一种常用的机器学习算法，经常用于训练模型以最小化损失函数。其基本思想是沿着函数的梯度方向（即函数变化最快的方向）进行迭代，以找到函数的局部最小值。
